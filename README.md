## Installation

Once you've downloaded the latest release, you must [install pygame.](https://www.pygame.org/wiki/GettingStarted#Pygame%20Installation)

```shell
python3 -m pip install -U pygame --user
```

To test the game, run:

```shell
python3 main.py
```
